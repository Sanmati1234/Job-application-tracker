# Job Application Tracker

An interactive web-based job application tracking tool that helps students and job seekers manage their job hunt with a drag-and-drop Kanban board and comprehensive dashboard statistics.

## Features

- **Kanban Board**: Drag and drop interface to move applications across different stages (Applied, Interviewing, Offered, Rejected)
- **Application Management**: Add notes, interview dates, and track application details
- **Dashboard Statistics**: View total applications, interview rates, offer rates, and stage counts
- **Filtering**: Filter applications by company or role
- **Responsive Design**: Works on desktop and mobile devices

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Chart.js for data visualization
- Google Fonts (Inter)

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/Sanmati1234/Job-application-tracker.git
   ```

2. Navigate to the project directory:
   ```
   cd Job-application-tracker
   ```

3. Start the local server:
   ```
   python -m http.server 3000
   ```

4. Open your browser and go to `http://localhost:3000`

## Usage

1. **Adding Applications**: Click the "Add Application" button to create new job applications
2. **Moving Applications**: Drag cards between columns to update their status
3. **Viewing Stats**: Check the dashboard at the top for application statistics
4. **Filtering**: Use the filter options to view applications by company or role
